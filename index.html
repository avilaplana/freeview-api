TVLIVE.IO

TV guide API for FREEVIEW and SATELLITE television in UK

The purpose of the API

The reason why I decided to start this open source project is because I wanted to develop a mobile application to show me the content of the tv and provide rate of that. I am aware that all the modern smart tvs provide really cool user interface showing all the the tv guide. But I personally struggle to find the right content and fitted to my taste. I really missed a good search based on genres, series, programs and very important, a really relaible rate indicating the level of that tv program.

API doc

You can find all the details of the API here: http://docs.tvlive.apiary.io

Software as a service

Everybody can use the service deployed in http://beta.tvlive.io by free.

Let's test it

List of roots (resources/endpoints to start navigating through the content ): 

$curl http://beta.tvlive.io
List of TV channels: 

$curl http://beta.tvlive.io/channels
List of TV channels with category 'kids': 

$curl http://beta.tvlive.io/channels/category/kids
List of TV channels with TV provider 'freeview': 

$curl http://beta.tvlive.io/channels/provider/freeview
List of all the channel TV providers:

$curl http://beta.tvlive.io/providers
List of all the channel TV categories:

$curl http://beta.tvlive.io/categories
TV content today in BBC One London: 

$curl http://beta.tvlive.io/tvcontent/channel/BBC+ONE+LONDON/today
TV content left today in BBC One London: 

$curl http://beta.tvlive.io/tvcontent/channel/BBC+ONE+LONDON/left
What's on in BBC One London: 

$curl http://beta.tvlive.io/tvcontent/channel/BBC+ONE+LONDON/current
TV series for FREEVIEW today: 

curl http://beta.tvlive.io/tvcontent/series/freeview/today
TV programs for FREEVIEW left today: 

curl http://beta.tvlive.io/tvcontent/program/freeview/left
What films for FREEVIEW are on TV now: 

curl http://beta.tvlive.io/tvcontent/film/freeview/current
TV program details: 

curl http://beta.tvlive.io/tvcontent/540c028aa3d9bd4ae35dbfbe
