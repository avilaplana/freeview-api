TVLIVE.IO

TV guide API for FREEEVIEW and SATELLITE television in UK

The purpose of the API

The reason why I decided to start this open source project is because I wanted to develop a mobile application to show me the content of the tv and provide rate of that. I am aware that all the modern smart tvs provide really cool user interface showing all the the tv guide. But I personally struggle to find the right content and fitted to my taste. I really missed a good search based on genres, series, programs and very important, a really relaible rate indicating the level of that tv program.

API doc

You can find all the details of the API here: http://docs.tvlive.apiary.io

Software as a service

Everybody can use the service deployed in http://beta.tvlive.io by free.

Let's test it

List of TV channels: 

$curl http://beta.tvlive.io/channels
List of genres for tv channels: 

$curl http://beta.tvlive.io/genres
List of TV channels by genre: 

$curl http://beta.tvlive.io/channels/kids
TV content today in BBC 1 London: 

$curl http://beta.tvlive.io/tvcontent/channel/BBC+1+LONDON/today
TV content left today in BBC 1 London: 

$curl http://beta.tvlive.io/tvcontent/channel/BBC+1+LONDON/left
What's on in BBC 1 London: 

$curl http://beta.tvlive.io/tvcontent/channel/BBC+1+LONDON/current>
Western TV content today: 

curl http://beta.tvlive.io/tvcontent/genre/western/today
Horror TV content left today: 

curl http://beta.tvlive.io/tvcontent/genre/horror/left
What documentaries are on TV now: 

curl http://beta.tvlive.io/tvcontent/genre/documentary/current
TV program details: 

curl http://beta.tvlive.io/tvcontent/540c028aa3d9bd4ae35dbfbe
